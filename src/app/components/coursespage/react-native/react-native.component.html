Заглиблення у подробиці розробки мобільних додатків, проектування, налагодження, доставки клієнтам.

Освітня програма торкнеться таких тем: організація й принципи побудови інтерфейсу користувача, прийоми роботи з графічними макетами проектів, взаємодія із API платформами, платформозалежні компоненти, PUSH-нонтифікації, організація коду, взаємодія із зовнішніми сервісами та інше.

  Початок курсу: 12 березня
  2 місяці, Вівторок - четвер з 18:00 до 21:00
  Годин практики: 48
  Мінімальні вимоги:

- Навики з розробки на JavaScript
- Бажаний досвід у React.js
- навички роботи з GIT
- Наявність власного ноутбуку для занять у аудиторії.

  Результат по закінченню курсу:

- Розробка мобільного додатку із повним функціоналом.
- Розуміння та проектування загальної структури додатку.
- Використання та налаштування нативних компонентів платформи.
- Взаємодія із зовнішніми сервісами.
- Повний цикл налагодження й доставки додатку у AppStore/Play Market
